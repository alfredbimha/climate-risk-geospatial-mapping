# Climate Risk Geospatial Analysis

## Research Question
Which regions face the greatest physical climate risk to economic assets?

## Methodology
**Language:** Python  
**Methods:** Spatial analysis, climate exposure scoring

## Data
NOAA climate data, EPA facility data

## Key Findings
Coastal and southern US regions show highest composite climate risk scores.

## How to Run
```bash
pip install -r requirements.txt
python code/project8_*.py
```
